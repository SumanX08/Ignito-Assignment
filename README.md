#  Bookstore Management App

A simple **MERN stack** project to demonstrate backend CRUD operations for a bookstore system.  

---

## Functionalities

- **Books Management**
  - Add, view, update, delete books
- **Authors Management**
  - Add, view, update, delete authors
- **Publishers Management**
  - Add, view, update, delete publishers
- **Routing**
  - to navigate between Books, Authors, and Orders

---

##  Tech Stack

- Node.js
- Express.js
- MongoDB with Mongoose
- React.js
- Axios for API calls
- React Router for navigation



## API Endpoints

- **Books**
   - *POST /books → Add a new book*
   - *GET /books → Get all books*
   - *PUT /books/:id → Update book details*
   - *DELETE /books/:id → Delete a book*

- **Authors**
   - *POST /authors*
   - *GET /authors*
   - *PUT /authors/:id*
   - *DELETE /authors/:id*

- **Publishers**
   - *POST /publishers*
   - *GET /publishers*
   - *PUT /publishers/:id*
   - *DELETE /publishers/:id*
